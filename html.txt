HyperText Markup Language(HTML5)
- HTML 기본구조 
<!DOCTYPE html>
<html>
  <head> <!--웹브라우저가 이문서를 인식하기 위한 부분-->
    <meta charset="UTF-8">
    <title>탭제목</title>
  </head>
  <body> <!--사용자가 화면에 보는 부분-->
  </body>
</html>

<h1>~<h6> : 문단제목 - 숫자가 작을수록 큰글자, 자동줄바꿈
<p> : 문단 , 자동 줄바꿈
<br> : 줄바꿈실행, 종료태그가 없다.
html에서 공백, 탭, 엔터는 모두 하나의 공백으로 이해
<hr> : 수평선보여줌, 종료태그가 없다.
<strong>, <b> : 굵은 글씨
<em>, <i> : 이텔릭체 (기울임글씨)
<ins>, <u> : 밑줄글씨
<del>, <s> : 삭제글씨 (글씨중앙에 줄줄)
- 리스트
<ol> -> <li> : 순서가 있는 리스트
  속성: type="" 1(숫자), a(소문자), A(대문자), i(로마소문자), I(로마대문자)
  start="숫자" - 시작번호
<ul> -> <li> : 순서가 없는 리스트 (문장앞에 점)
- 표만들기 (table)
기본구조 (2x2)
<table> <!-- 테이블을 만드는 태그-->
  <tr>  <!-- 한 줄을 의미 -->
    <td></td> <!-- 한 셀을 의미, 굵은글씨를 나타내는 <th> -->
    <td></td>
  </tr>
  <tr>
    <td></td>
    <td></td>
  </tr>
</table>
- <td>, <th>태그에 넣을 수있는 속성
  rowspan="" - 행을 합치는 기능 : 다음행에 셀하나를 뺀다
  colspan="" - 열을 합치는 기능 : 다음셀을 뺀다.
<img> : 이미지를 넣는 태그에, 종료태그가 없다.
  속성 ; src="" - 이미지의 경로와 이름
         alt="" - 이미지가 어떠한 이유로 불러오지 못했을때 나타나는 글씨
         width="" - 이미지의 가로폭, 예) 100%: 웹브라우저 가로사이즈에 자동으로 맞춘다.
<audio> : 음악
<video> : 영상
  속성 ; src="" - 음악또는 영상의 경로와 이름
         controls - 조작하는 창
<a> : 다른페이지로 이동하는 태그, 시작태그와 종료태그 사이에 컨텐츠(텍스트, 이미지, 영상..)가 있어야 합니다.
  속성 ; href="" - 이동할 주소를 적는다.
  사용예) <a href="www.naver.com">네이버</a>
<div> : 블럭을 만들어주는 태그
5장> 폼태그 (form)
<form> - 서버로 사용자의 입력을 전달
  속성; method="" : 입력값의 전송방식 (GET, POST)
        action="" : 전송할 주소
<filedset> : 그룹화
<legend> : 그룹의 이름
<label> : 폼요소에 이름을 붙이는 태그
  속성 : for="" : 폼요소의 id속성값을 매칭시킨다.
<input> : 사용자 입력을 위한 대표적인 태그, 종료태그없음
  속성 : type=""
    - text : 한줄 글씨 입력
    - password : 글씨 입력시 * or 점으로 나타남
    - email : 이메일형식으로 입력
    - date : 날짜입력
    - time : 시간입력
    - check-box : 여러개 선택 입력
    - radio : 라디오버튼 (여러개중 하나 선택)
    - submit : 버튼을 눌렀을때 action에 적힌 주소로 이동
    - button : 일반 버튼
    - reset : 버튼을 눌렀을때 사용자 입력을 클리어
    - hidden : 화면에는 나타나지 않지만 특정값을 서버에 전송하고 싶을때 사용
    - file : 화일을 전송하고자 할때 사용하는 타입
  속성 : required - 필수입력
  속성 : readonly - 쓰기불가 - 화면에 보여주고자 할때
<textarea> : 여러줄을 쓰고자 할때 사용
  속성 : cols="" - 가로글자수
         rows="" - 행의 수
<select>, <option> : 드랍다운메뉴
<button> : 버튼을 만들때
  속성 : type="" : submit, button, reset (input태그에서 설명), 폼 태그안의 버튼은 기본type이 submit이다.
