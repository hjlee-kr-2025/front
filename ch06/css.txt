CSS (Cascading Style Sheet) - CSS3 level3
- Style : 꾸민다.

**스타일을 만드는 기본형식**
선택자 {속성1: 속성값; 속성2: 속성값; ....}
- 선택자는 스타일을 입힐 요소
선택자 {
  속성1:속성값1;
  속성2:속성값2;
}

** 스타일을 만들어주는 방법
1. 웹브라우저의 디폴트 스타일
2. 외부파일을 사용하는 방법
3. 내부에서 <style> 태그를 이용하여 구성
4. 인라인 스타일 - 태그안에 style속성을 이용하여 구성

스타일을 만드는 우선순위
1. 위의 스타일방법의 반대로 우선순위를 가진다.
  인라인->내부->외부->브라우저 디폴트
2. 같은 우선순위일때 - 마지막에 적힌 내용으로 만들어진다.

선택자 종류에 따른 우선순위
1. !important 선언되어있는 스타일
2. 인라인스타일
3. id선택자
4. class선택자
5. 타입(태그)선택자자
