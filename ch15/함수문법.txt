함수를 정의하는 방법

(기본문법) - 함수이름: 영문자, 숫자, _ 사용, camel표기법
function 함수이름(변수이름1, 변수이름2, ...) { //()안에 아무값도 안들어 가도 됩니다.
  함수가 처리해야할 코드들...
}

함수사용방법
함수이름(변수에 들어가는 값1, ...);
변수에 들어가는 값 개수는 함수에서 정의해준 개수대로 적는다.

함수정의2)
function 함수이름(변수이름1, 변수이름2) {
  처리문;
  return 값;
}

함수사용2)
변수이름 = 함수이름(변수값1, 변수값2);

함수의 4가지 유형 예시
1) 매개변수(파라메타;parameta) x, 리턴값(return) x
function addNumber() {
  처리문;
}
2) 매개변수 o, 리턴값 x
function addNumber(num1, num2) {
  처리문;
}
3) 매개변수 x, 리턴값 o
function addNumber() {
  처리문;
  return 값;
}
4) 매개변수 o, 리턴값 o
function addNumber(num1, num2) {
  처리문;
  return 값;
}
