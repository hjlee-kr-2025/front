Javascript 배운내용정리

HTML코드에 아래와 같이 작성해서 사용합니다.
<script>
// 자바스크립트 코드 구현
// 한줄주석은 이렇게 사용합니다. 
/*
여러줄 주석은 이렇게 사용합니다.
*/
</script>

변수 : 데이터를 저장, 변경할때 사용하는 이름
변수는 사용전에 선언을 해야합니다.
var 변수이름;
변수이름은 영문자, 숫자, _ 정도만 사용해서 만들어 줍니다. 
변수이름을 만들때 의미가 있게 만들고, camel표기법을 사용하면 좋습니다.

변수의 자료형
숫자형 - 숫자만 사용합니다. (정수 또는 실수)
문자형 - 데이터를 '' 또는 "" 로 묶어서 사용합니다.
논리형 - true or false

내장함수(자바스크립트에 구현되어있는 함수)
alert("알림내용"); -> 화면에 알림메시지를 보여줄때 사용
confirm("질문의 내용에 확인/취소 를 받을때");
prompt("값을 입력받을때 사용");

개발자가 코드를 확인할때 가장 많이 사용하는 코드
console.log(확인내용);

명령문 끝에 ;(세미콜론)을 붙인다.

조건문 / 반복문
조건문
===== if문의 여러가지 유형 =====
1)
if (조건1) {
  조건1이 참일때 실행
}
2)
if (조건1) {
  조건1이 참일때 실행
}
else {
  조건1이 참이 아닐때 실행행
}
3)
if (조건1) {
  조건1이 참일때 실행
}
else if (조건2) {
  조건1이 참이 아니고, 조건2가 참일때 실행행
}
else {
  조건1이 참이 아니고, 조건2도 참이 아닐때 실행행
}

===== switch ~ case ~ break =====
switch (변수) {
  case A:
    변수가 A와 같으면 실행
    break; -> break를 만나면 switch 문을 빠져 나옵니다.
  case B:
    변수가 B와 같으면 실행
    break;
    ....
  default:
    이곳은 case문이 만족되는 것이 없으면 실행되는데
    사용할 필요가 없으면 삭제해도 관계없습니다.
}

반복문 : for, while, do~while
for문법
for (1. 초기화 ; 2. 조건 ; 4.초기화에 사용된 변수를 증감) {
  3.조건이 참일때 처리문
}
처리순서 : 1->2->3->4->2->3->4->2...

while 문법
while (조건) {
  조건이 참일때 계속해서 반복실행
}
